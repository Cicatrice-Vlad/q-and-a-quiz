# Q&A Quiz

## Description

The motivation for this app came from finding out how Java behaves when interracting with it on a quiz game.
The project was built on the hopes that can answer a few question about the AI's logic and JavaScript functions that can be easily understood once learnt.
The project solves an issue that stands between the user and the AI device, which is communication and it works!
The project teaches us all that the app is working due to the AI system that JavaScript is operating and can function on a communicational level.
